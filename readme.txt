This plugin fixes Contact Form 7's Recaptcha v3 so it only appears on pages with contact forms.

== Description ==

This plugin ensures that Contact Form 7's Recaptcha v3 only appears on pages with contact forms, improving performance and reducing unnecessary load.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/show-recaptcha-where-nedeed` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Show Recaptcha screen to configure the plugin

== Frequently Asked Questions ==

= How do I enable or disable the Recaptcha functionality? =

Go to Settings->Show Recaptcha and use the checkbox to enable or disable the Recaptcha functionality.
